Import Content

Upload YAML files to import blog posts or reviews into the system.

Import Requirements

  • Files must be in YAML format (.yaml or .yml extension)
  • Blog posts must contain:
    • title - Blog post title (required)
    • content - Blog post content (required)
    • author - Author name (optional, defaults to "Admin")
    • date - Publication date (optional, defaults to current time)
    • tags - List of tags (optional)
    • metaDescription - SEO meta description (optional)
  • Reviews must contain:
    • title - Review title (required)
    • content - Review content (required)
    • productBrand - Product brand name (required)
    • productName - Product name (optional)
    • rating - Rating 1-5 (optional, defaults to 5.0)
    • pros - Product advantages (optional)
    • cons - Product disadvantages (optional)
    • conclusion - Review conclusion (optional)
  • Missing required fields will show specific error messages
  • File size limit: Maximum 5MB
  • Invalid YAML syntax will be detected and reported

YAML Format Examples

title: "Best GPS Pet Trackers for Dogs in 2025
author: "John Smith"
date: "2025-01-15 10:30:00"
tags:
  - "pet tracker"
  - "dogs"
  - "GPS"
metaDescription: "Comprehensive guide to the best GPS pet trackers for dogs in 2025"
content: |
  In this comprehensive guide, we'll explore the top GPS pet trackers
  available for dogs in 2025. These devices help you keep track of
  your furry friends and ensure their safety.
  
  ## Key Features to Consider
  - Battery life
  - GPS accuracy
  - Water resistance
  - Mobile app quality
